The Astronomy Logbook Project

Logbooks for Amateur Astronomers!

This is a project aimed at creating freely distributable, high-quality logbooks for amateur astronomers, fully complete with star charts and reference images.

Features

Basic Data

A table containing basic data about the object is available for quick reference.

Observability Icons

Some logbooks feature icons that tell you if the object is visible under light-polluted skies, with binoculars, etc.

Star Chart

One or more (at different zoom levels) annotated star charts help you find the object.

DSS Imagery

Imagery from the Digitized Sky Survey helps you know what you're looking at.

Logging form

With fields for logging your observations.
